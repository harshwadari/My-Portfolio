## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/harshwadari/portfolio.git
```

Navigate to the project:

```bash
cd portfolio
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
